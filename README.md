# Installation 
# git insttallation   : https://git-scm.com/downloads
# composer            : https://getcomposer.org/download/
# clone project       : git clone https://github.com/bangtiray/next-io-jastan.git
# cd next-io-jastan
# running on terminal 
  1. composer install
  2. cp .env.example .env   and set your Dabatabase please see https://laravel.com/docs/5.7/database
  3. php artisan key:generate
  4. php artisan migrate
  5. php artisan serve
  6. open and running on browser
